Stampa

Familia tipográfica de íconos para trabajar en conjunto con Pyxis.

Stampa es un set de 89 íconos, que utlizando las bases de Font Awesome, permite una serie de usos en relación con el framework Pyxis.
En la versión 2.0.0 de Stampa, se introducen los íconos lineales como una variante, usando la terminación -l para su nombramiento.
